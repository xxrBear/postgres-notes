# 运算符参考

**算术运算符**

| 运算符 | 含义 | 示例    | 结果  |
| ------ | ---- | ------- | ----- |
| `+`    | 加法 | `2 + 3` | `5`   |
| `-`    | 减法 | `5 - 2` | `3`   |
| `*`    | 乘法 | `4 * 2` | `8`   |
| `/`    | 除法 | `7 / 2` | `3.5` |
| `%`    | 取余 | `7 % 3` | `1`   |
| `^`    | 乘方 | `2 ^ 3` | `8`   |

**比较运算符**

| 运算符       | 含义     | 示例     | 结果   |
| ------------ | -------- | -------- | ------ |
| `=`          | 等于     | `5 = 5`  | `true` |
| `<>` 或 `!=` | 不等于   | `5 <> 4` | `true` |
| `<`          | 小于     | `3 < 5`  | `true` |
| `<=`         | 小于等于 | `3 <= 3` | `true` |
| `>`          | 大于     | `7 > 2`  | `true` |
| `>=`         | 大于等于 | `5 >= 5` | `true` |

**字符串运算符**

| 运算符  | 含义                     | 示例               | 结果   |
| ------- | ------------------------ | ------------------ | ------ |
| `LIKE`  | 模式匹配（大小写敏感）   | `'abc' LIKE 'a%'`  | `true` |
| `ILIKE` | 模式匹配（大小写不敏感） | `'ABC' ILIKE 'a%'` | `true` |
| `~`     | POSIX 正则匹配           | `'abc' ~ 'a.*'`    | `true` |
| `!~`    | 正则不匹配               | `'abc' !~ '^z'`    | `true` |

**集合 / 范围运算符**

| 运算符        | 含义             | 示例                                | 结果   |
| ------------- | ---------------- | ----------------------------------- | ------ |
| `IN`          | 是否属于集合     | `3 IN (1,2,3)`                      | `true` |
| `NOT IN`      | 是否不属于集合   | `4 NOT IN (1,2,3)`                  | `true` |
| `BETWEEN`     | 范围包含         | `5 BETWEEN 1 AND 10`                | `true` |
| `NOT BETWEEN` | 范围不包含       | `0 NOT BETWEEN 1 AND 10`            | `true` |
| `<@`          | 元素是否在范围内 | `5 <@ int4range(1,10)`              | `true` |
| `&&`          | 范围是否有重叠   | `int4range(1,5) && int4range(4,10)` | `true` |

**全文搜索运算符**

| 运算符 | 含义                     | 示例                                    | 结果   |
| ------ | ------------------------ | --------------------------------------- | ------ |
| `@@`   | 匹配                     | `'fat cat'::tsvector @@ 'cat'::tsquery` | `true` |
| `@@@`  | 高级匹配（不同版本扩展） | 用于自定义扩展                          | -      |
